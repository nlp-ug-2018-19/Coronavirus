About our project

This website has been created as a group project for a Web Development course at University of Gdansk under the supervision of dr Marcin Fortuna.

The project was created to adress the presssing needs of the society in the peculiar times of Coronavirus and social distancing. To help people adapt to the unusual circumstances we decided to provide a solution for citizens #stayingathome who are desperately seeking for entartainment during lockdown.
We created a base of possible hobbies which people might like to take up and implemented the state of the art chatbot programme which can help them choose the right activity based on their previous experience.

The main technologies used in this project were:
  • HTML
  • CSS
  • Web Speech API
  • Bootstrap
  • Ingenious custom chatbot
